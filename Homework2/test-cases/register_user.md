# Пользователь может зарегистрироваться используя email

## Предусловие

1. Открыть https://demo.realworld.io/

## Шаги

1. Кликнуть на кнопку **Sign Up** в хедере
2. Кликнуть на поле **Username**
3. Ввести `{user_name}` в поле **Username**
3. Кликнуть на поле **Email**
4. Ввести `{email}` в поле **Email**
5. Кликнуть на поле **Password**
6. Ввести `{password}` в поле **Password**
7. Кликнуть на кнопку **Sign up** 

## Ожидаемый результат

1. В профиле отобразилось имя пользователя `{user_name}`


# Памятка:

* `{user_name}`
    * маска `[0-9a-zA-Z_]{5, 10}`
    * ранее не использовалось
* `{email}`
    * валидный
    * ранее не использовался
* `{password}` — маска `[0-9a-zA-Z_]{6, 16}`
